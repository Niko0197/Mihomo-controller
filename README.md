# Mihomo Controller

Красивая и функциональная веб-панель управления ядром **Mihomo (Clash)** на роутерах Keenetic с установленным Entware (например, через проект **XKeen**).

Выполнена в современном адаптивном дизайне **Material 3 (Dark Theme)**. Полностью написана на чистом JavaScript / Node.js **без каких-либо внешних NPM-зависимостей**, что гарантирует максимальную производительность, безопасность и нулевое потребление лишних ресурсов роутера.

---

## Основные возможности

1. **📊 Трафик в реальном времени**:
   - Живой график скорости загрузки и отдачи (Chart.js) за последние 60 секунд.
   - Раздельный накопительный подсчёт потреблённых данных через VPN и напрямую (DIRECT) на основе дельта-анализа соединений.
2. **🔌 Активные соединения**:
   - Таблица текущих сетевых сокетов с подсчётом скорости передачи по каждому сокету в реальном времени.
   - Умное сокращение длинных имен хостов и правил с всплывающими подсказками (tooltips) при наведении.
   - Возможность моментально разорвать любое соединение или закрыть все сокеты разом.
3. **📋 Системные логи ядра**:
   - Полноценный real-time стриминг логов с разделением по уровням (`Debug`, `Info`, `Warning`, `Error`) на независимых вкладках.
   - **Persistence**: Фоновый сбор и кэширование логов (до 1000 строк). При переключении вкладок логов история не теряется, а продолжает накапливаться.
   - **Smart Scroll Lock**: Индивидуальный автоскролл для каждой вкладки логов — если вы прокрутили список вверх, новые логи не будут дёргать экран, но продолжат загружаться.
   - Полнотекстовый живой поиск по ключевым словам.
4. **🛰️ Трассировка маршрута (Trace Route)**:
   - Инструмент тестирования: введите домен, и панель симулирует прохождение по цепочке правил `config.yaml` (включая рекурсивные составные правила `OR` / `AND`), покажет разрешённые IP, GeoIP страну и итоговый маршрут.
5. **📝 Редактор YAML**:
   - Полноценный встроенный редактор конфигурации `config.yaml` (CodeMirror) с автоматическим бэкапом при сохранении.
6. **🔌 Импорт и подписки**:
   - Парсер и импорт прокси-ссылок форматов `vless://`, `ss://`, `trojan://` с автоматической привязкой к выбранным группам.
   - Менеджер подписок (Proxy Providers) с возможностью обновлять и редактировать провайдеров прямо из интерфейса.
7. **⚡ Пинг-тест**:
   - Быстрый замер задержки выбранного узла или массовый пинг-тест всех добавленных прокси-серверов в 5 параллельных потоков.
8. **🌉 Tor-мосты**:
   - Автоматический импорт актуальных рабочих мостов Tor из репозитория Delta-Kronecker.

## Установка на Keenetic (Entware)

Для установки откройте SSH-консоль вашего роутера и выполните одну команду:

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ВАШ_GITHUB_ЛОГИН/mihomo-controller/main/install.sh)"
```

> **Примечание**: Перед установкой не забудьте заменить `ВАШ_GITHUB_ЛОГИН` в URL команды (и внутри файла `install.sh`) на имя вашего аккаунта на GitHub, куда вы загрузили файлы.

Панель автоматически установит Node.js (если его нет), скопирует файлы в `/opt/root/mihomo-controller/`, пропишет службу автозапуска в Entware и запустит веб-интерфейс на порту **4000**:
[http://192.168.1.1:4000](http://192.168.1.1:4000)
