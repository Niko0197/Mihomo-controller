Красивая и функциональная веб-панель управления ядром **Mihomo (Clash)** на роутерах Keenetic с установленным Entware (например, через проект **XKeen**).

Выполнена в современном адаптивном дизайне **Material 3 (Dark Theme)**. Полностью написана на чистом JavaScript / Node.js **без каких-либо внешних NPM-зависимостей**, что гарантирует максимальную производительность, безопасность и нулевое потребление лишних ресурсов роутера.

---

## ✨ Основные возможности

### 🛠️ Быстрые правила маршрутизации
- **Динамическое добавление правил**: Добавление собственных правил (домены, IP, подсети) прямо из веб-панели.
- **Изменение направления «на лету»**: Мгновенное перенаправление трафика (смена прокси-группы или переключение на DIRECT) через удобный выпадающий список для каждого правила.
- **Интеграция без перезапуска**: Правила применяются в памяти ядра Mihomo мгновенно без разрыва активных соединений и перезапуска службы.
- **Защита системных правил**: Визуальное разделение правил на статические системные (защищены замочком `🔒`) и пользовательские динамические (доступны изменение и удаление `🗑️`).
- **Безопасность**: Полная валидация IP-адресов, масок подсетей, синтаксиса доменов и автобэкап перед сохранением конфигурации.

### 🔄 Обновление ядра Mihomo и веток панели
- **Управление ядром Mihomo**: Обновление и откат версии ядра Mihomo прямо из веб-интерфейса панели.
- **Интерактивный список релизов**: Просмотр последних 15 релизов ядра с GitHub, включая подробную историю изменений (What's Changed) для каждого релиза.
- **Автоопределение архитектуры процессора**: Автоматическое сопоставление архитектуры роутера (например, `arm64`, `arm32v7`, `mipsle` с поддержкой softfloat, и др.) для скачивания правильного `.gz` архива.
- **Безопасная установка**: Потоковая распаковка архива через встроенный `zlib` (для экономии ОЗУ роутера) с автоматическим бэкапом старого файла ядра и мгновенным откатом при сбое запуска нового бинарника.
- **Управление версиями панели**: Удобное переключение между ветками обновлений панели (`main` и `Dev`) и откат к конкретному коммиту из истории последних 15 коммитов.

### 📟 Встроенный мониторинг ресурсов роутера (в сайдбаре)
- **Компактная интеграция**: Блок мониторинга встроен прямо в нижнюю часть статус-панели сайдбара.
- **Основные метрики**: Измерение нагрузки на процессор (ЦП), потребления оперативной памяти (ОЗУ) роутера и его текущей температуры.
- **Живой график истории**: Интерактивный график истории нагрузки (ЦП, ОЗУ, температура) за последние 60 секунд.
- **Плавная iOS-style анимация**: Раскрытие панели (accordion transition по `max-height`/`opacity` на кривой `cubic-bezier(0.25, 1, 0.5, 1)`) с плавным вращением стрелки-индикатора на 90 градусов.
- **Защита от ложных кликов**: Клики по графику и элементам внутри панели игнорируются, закрытие происходит строго по клику на область заголовка.

### 📊 Трафик и скорость в реальном времени
- **Посекундное обновление**: Обновление скоростей устройств и графиков трафика каждую секунду для максимальной отзывчивости интерфейса.
- **Живой график**: График скорости загрузки и отдачи (Chart.js) за последние 60 секунд.
- **Дельта-анализ**: Раздельный накопительный подсчёт потреблённых данных через VPN и напрямую (DIRECT).

### 🔌 Активные соединения
- **Мониторинг сокетов**: Таблица текущих сетевых сокетов с посекундным обновлением скоростей передачи.
- **Умный интерфейс**: Сокращение длинных имен хостов и правил с подробными всплывающими подсказками (tooltips) при наведении.
- **Управление сокетами**: Возможность моментально разорвать любое соединение или закрыть все сокеты разом.

### 📋 Системные логи ядра
- **Real-time стриминг**: Стриминг логов с разделением по уровням (`Debug`, `Info`, `Warning`, `Error`) на независимых вкладках.
- **Persistence**: Фоновый сбор и кэширование логов (до 1000 строк). При переключении вкладок логов история не теряется, а продолжает накапливаться.
- **Smart Scroll Lock**: Индивидуальный автоскролл для каждой вкладки логов — если вы прокрутили список вверх, новые логи не будут дёргать экран, но продолжат загружаться.
- Полнотекстовый живой поиск по ключевым словам.

### 👥 Менеджер клиентов (Устройства)
- **Список устройств**: Подробный список подключенных устройств с отображением их текущей сетевой активности и скорости в реальном времени.
- **Правила маршрутизации**: Управление правилами для конкретных устройств в сети по IP-адресам.
- **Назначение прокси**: Назначение устройств на прокси-группы или DIRECT с автоматической перезаписью конфигурационного файла.

### ⚡ Премиальные анимации и оптимизация ОЗУ
- **Тактильный отклик кнопок**: Асимметричная пружинная анимация нажатия кнопок навигации и управления (`:active`): мгновенное сжатие за `0.06s` и плавный возврат в исходный масштаб за `0.18s`.
- **Эффективное использование ОЗУ**: Ограничение потребления оперативной памяти Node.js на уровне ~35-48MB RSS благодаря оптимизированным флагам работы с памятью V8 на роутере.

### 🛰️ Трассировка маршрута (Trace Route)
- Инструмент тестирования: введите домен, и панель симулирует прохождение по цепочке правил `config.yaml` (включая рекурсивные составные правила `OR` / `AND`), покажет разрешённые IP, GeoIP страну и итоговый маршрут.

### 📝 Редактор YAML
- Полноценный встроенный редактор конфигурации `config.yaml` (CodeMirror) с автоматическим бэкапом при сохранении.

### 🔌 Импорт и подписки
- Менеджер подписок (Proxy Providers) с возможностью обновлять и редактировать провайдеров прямо из интерфейса.
- Парсер и импорт прокси-ссылок форматов `vless://`, `ss://`, `trojan://` с автоматической привязкой к выбранным группам.

### ⚡ Пинг-тест
- Быстрый замер задержки выбранного узла или массовый пинг-тест всех добавленных прокси-серверов в 5 параллельных потоков.

### 🌉 Tor-мосты
- Автоматический импорт актуальных рабочих мостов Tor из репозитория Delta-Kronecker.

## 🚀 Установка

Для первой установки откройте SSH-консоль вашего роутера и выполните одну команду:

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Niko0197/Mihomo-controller/main/install.sh)"
```

Скрипт автоматически:
- ✅ Установит Node.js, curl и tar (если их нет)
- ✅ Скачает файлы с GitHub в `/opt/root/mihomo-controller/`
- ✅ Создаст службу автозапуска в Entware
- ✅ Запустит веб-интерфейс на порту **4000**

После установки панель доступна по адресу: **http://192.168.1.1:4000**

---

## 🔄 Обновление

Для обновления до последней версии выполните ту же команду:

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Niko0197/Mihomo-controller/main/install.sh)"
```

Скрипт **автоматически определит**, что панель уже установлена, и выполнит обновление:
- ✅ Обновит только код приложения (`server.js`, `updater.js`, `public/` и др.)
- ✅ **Сохранит** все пользовательские данные:
  - `clients_db.json` — база клиентов
  - `tor_bridges.json` — Tor-мосты
  - Логи, бэкапы конфига и другие пользовательские файлы
- ✅ Перезапустит службу

Также можно явно указать режим:

```bash
# Принудительное обновление (без переустановки зависимостей):
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Niko0197/Mihomo-controller/main/install.sh)" -- update

# Принудительная чистая установка (удаляет всё):
sh -c "$(curl -fsSL https://raw.githubusercontent.com/Niko0197/Mihomo-controller/main/install.sh)" -- install
```
